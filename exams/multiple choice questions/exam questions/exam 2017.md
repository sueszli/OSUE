# 1. Frage:

Was trifft auf ein Makefile zu?

- Ein Makefile ist ein Shell script.
- Ein Makefile enthält "dependencies".
- Links vom Doppelpunkt stehen "targets", rechts davon die "dependencies" und darunter die "commands".
