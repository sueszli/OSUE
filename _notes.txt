Exercise 3

Client requests file from server:

| GET <path> HTTP/1.1                        
| Host: www.name.com                         
| <...>                                      <--- arbitrary number of header fields
|                                            <--- no request body

Server transfers file to client:

| HTTP/1.1 200 OK
| Date: <date>
| Content-Type: text/html; charset=UTF-8
| Content-Length: <length>
| Last-Modified: <date>
| Connection: close
|
| <response body / requested file>

ALL STRING FUNCTIONS IN C:

copy from src to dst:
      char *strdup(const char *s);
      char *stpcpy(char *dest, const char *src);
      char *strcpy(char *dest, const char *src);
      char *strncpy(char *dest, const char *src, size_t n);
      size_t strxfrm(char *dest, const char *src, size_t n);
      int asprintf(char **strp, const char *fmt, ...);

append with string:
      char *strncat(char *dest, const char *src, size_t n);

shuffle:
      char *strfry(char *string);

get length:
      size_t strlen(const char *s);

compare 2 strings:
      int strncmp(const char *s1, const char *s2, size_t n);
      int strcasecmp(const char *s1, const char *s2);
      int strncasecmp(const char *s1, const char *s2, size_t n);
      char *strcat(char *dest, const char *src);
      int strcmp(const char *s1, const char *s2);
      int strcoll(const char *s1, const char *s2);

---

first / last index of char:
      char *index(const char *s, int c);
      char *strchr(const char *s, int c);
      char *rindex(const char *s, int c);
      char *strrchr(const char *s, int c);

first index of substring:
      char *strstr(const char *haystack, const char *needle);

first index of one of chars:
      char *strpbrk(const char *s, const char *chars);

count legal chars:
      size_t strspn(const char *s, const char *chars);  // count legal 
      size_t strcspn(const char *s, const char *chars); // count non legal

split string:
      char *strsep(char **stringp, const char *delim); 
      char *strtok(char *s, const char *delim);
