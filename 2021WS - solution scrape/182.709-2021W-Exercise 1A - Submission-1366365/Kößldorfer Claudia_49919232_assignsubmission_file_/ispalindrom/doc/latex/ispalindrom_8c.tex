If -\/o is provided output will be written in the given output file otherwise it will be written to stdout. An arbitrary number of input files can be given, which will be read from. If none are provided the program reads from stdin. If -\/s is provided whitespaces will be ignored for the check. If -\/i is provided the check will not differentiate between upper and lower case letters.
