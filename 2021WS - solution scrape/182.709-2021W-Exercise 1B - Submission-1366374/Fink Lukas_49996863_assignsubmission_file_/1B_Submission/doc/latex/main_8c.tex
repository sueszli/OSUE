The program starts here. It should be called like 'mycompress \mbox{[}-\/o outfile\mbox{]} \mbox{[}input\-File...\mbox{]}'. 

Program entry point.

It can handle an optional outfile as well as several input\-Files. If the option -\/o together with an outfile is not used, it writes to stdout. If it is called with no input files, it reads from stdin.
